\doxysubsection{Detailed Description}
This class used to render text as part of a \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{Symbol\+Collection}}. 

\begin{DoxyVerb}Text have a text string, font size, anchor location, and anchoring mode.
\end{DoxyVerb}
 Basic styling such as stroke, color are defined in the superclass \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}}.

On a label the \char`\"{}stroke\char`\"{} refer to the outside of the of the letter, and the \char`\"{}fill\char`\"{} refers to the inside of the letters. In most case, you want no stroke but a fill.

\mbox{\hyperlink{classbridges_1_1base_1_1_text}{Text}} is placed by defining an anchor position and what that position is for the text (is it the middle? is it the top left?). Check the tutorial for details.

An example tutorial can be found at \href{https://bridgesuncc.github.io/tutorials/Symbol_Collection.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Symbol\+\_\+\+Collection.\+html}}
\end{DoxySeeAlso}
