\doxysubsection{Detailed Description}
This class is used to represent cells in \mbox{\hyperlink{classbridges_1_1base_1_1_game_grid}{Game\+Grid}} in BRIDGES. It is part of the Game\+API. Each cell has a foreground color, background color, and a symbol shape.
