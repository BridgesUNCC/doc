\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1_shakespeare}{Shakespeare}} Data source object containing sonnets, poems and plays. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set.

One would not normally create an object of this type but rather get one by calling \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_af2b6e5df074d0a689e8a992b1bea6f3e}{bridges\+::connect\+::\+Data\+Source\+::get\+Shakespeare\+Data()}}

Refer to the tutorial on how to use this dataset\+: \href{https://bridgesuncc.github.io/tutorials/Data_Shakespeare.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Data\+\_\+\+Shakespeare.\+html}}
