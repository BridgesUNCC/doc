\doxysubsection{Detailed Description}
This class represents an actor-\/movie pair object, to be used with the IMDB actor-\/movie dataset. 

A user typically does not create an \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1_actor_movie_i_m_d_b}{Actor\+Movie\+IMDB}} object.
