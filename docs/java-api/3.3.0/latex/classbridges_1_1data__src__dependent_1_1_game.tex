\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1_game}{Game}} object, used along with the Games data source. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set.

One would not normally create an object of this type, but rather obtain them through a call to \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_ab3744c6b103281724bdd832da0924312}{bridges\+::connect\+::\+Data\+Source\+::get\+Game\+Data()}}.

Each game has a title, platform on which it can be played, rating and a list of genres

Refer to tutorial examples on how to use this dataset\+: \href{https://bridgesuncc.github.io/tutorials/Data_IGN_Games.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Data\+\_\+\+IGN\+\_\+\+Games.\+html}}
