\doxysubsection{Detailed Description}
This class defines a rectangle and is part of the \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{bridges\+::base\+::\+Symbol\+Collection}}. 

\begin{DoxyVerb}A rectangle has height and width, and a location (rectangle center)
\end{DoxyVerb}
 Basic styling such as stroke, fill, color are defined in the superclass \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}}.
