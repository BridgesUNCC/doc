\doxysubsection{Detailed Description}
This class defines a circle and is part of the \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{bridges\+::base\+::\+Symbol\+Collection}}. 

\begin{DoxyVerb}A circle has a center and radius
\end{DoxyVerb}
 Basic styling such as stroke, fill, color are defined in the superclass \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}}.
