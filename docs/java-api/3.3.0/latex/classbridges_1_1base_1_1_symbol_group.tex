\doxysubsection{Detailed Description}
This class defines a group of Symbols and is part of the \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{bridges\+::base\+::\+Symbol\+Collection}}. 

One would use \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_group}{Symbol\+Group}} to bundle together a set of \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}} that can be used and placed at one. For instance, one could create a group representing a house and then place that house in a scene.

Setting properties of groups may have surprising consequences. For instance setting the opacity of a \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_group}{Symbol\+Group}} will apply a multiplicative opacity to all the objects in the group, but setting an fill color will only impact the \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}} in the \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_group}{Symbol\+Group}} that do not have a defined fill color.
