\doxysubsection{Detailed Description}
This class defines a polygon and can be used as a part of a \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{Symbol\+Collection}}. 

\begin{DoxyVerb}A polygon has a sequence of 2D points (x, y coordinate pairs)
\end{DoxyVerb}
 Basic styling such as stroke, fill, color are defined in the superclass \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}}.
