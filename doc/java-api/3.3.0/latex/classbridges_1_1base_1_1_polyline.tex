\doxysubsection{Detailed Description}
This class defines a polyline and is used as part of a \mbox{\hyperlink{classbridges_1_1base_1_1_symbol_collection}{Symbol\+Collection}}. 

\begin{DoxyVerb}A polyline has a sequence of points (x, y coordinate pairs)
\end{DoxyVerb}
 Basic styling such as stroke and fill are defined in the superclass \mbox{\hyperlink{classbridges_1_1base_1_1_symbol}{Symbol}}.

Note that since a \mbox{\hyperlink{classbridges_1_1base_1_1_polyline}{Polyline}} is typically an open shape one usually want a fill that is transparent.
