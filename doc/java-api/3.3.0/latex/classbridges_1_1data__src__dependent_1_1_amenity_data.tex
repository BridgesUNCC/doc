\doxysubsection{Detailed Description}
Class that hold Open Street Map Amenity Data. 

This class holds the data for all the amenities requested by the user

Usually this class is not created by the user but returned by \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_aa25f4d7521aa1ad788b60816eb2088db}{bridges\+::connect\+::\+Data\+Source\+::get\+Amenity\+Data()}}

Check out the tutorial on getting amenity data at \href{https://bridgesuncc.github.io/tutorials/Data_Amenity.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Data\+\_\+\+Amenity.\+html}}
