\doxysubsection{Detailed Description}
Stores the meta data of a book as stored by project Gutenberg. 

This object stores the meta data of a book such as its title, author and genre. The meta data come from \href{https://www.gutenberg.org/}{\texttt{ https\+://www.\+gutenberg.\+org/}}.

Objects of this type are typically not constructed by the user but returned by a call to our Gutenberg API such as \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_a7f27778f3a8fcd01a2504b19d35eda14}{bridges\+::connect\+::\+Data\+Source\+::get\+Gutenberg\+Book\+Meta\+Data()}} or \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_a1f0ec9eb03a7e3631146415c317a1694}{bridges\+::connect\+::\+Data\+Source\+::get\+AGutenberg\+Book\+Meta\+Data(int)}}.

The object does not contain the text of the book itself. Though it can be obtained using \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_ae6a17bd40228b89350fa186a2caad292}{bridges\+::connect\+::\+Data\+Source\+::get\+Gutenberg\+Book\+Text()}}

A tutorial of how to use the Gutenberg data in BRIDGES is available\+: \href{https://bridgesuncc.github.io/tutorials/Data_Gutenberg.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Data\+\_\+\+Gutenberg.\+html}} \doxysubsection*{Public Member Functions}
