\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1_song}{Song}} object, used along with the Songs data source (using the Genius API. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set. The \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1_song}{Song}} object is typically obtained from calling \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_aeccb689aeacc094b0621ab5564849fa6}{bridges\+::connect\+::\+Data\+Source\+::get\+Song\+Data()}} or \mbox{\hyperlink{classbridges_1_1connect_1_1_data_source_a935ea2005b7de3cb22c1e55027b81460}{bridges\+::connect\+::\+Data\+Source\+::get\+Song()}}.
